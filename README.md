# Generador de Calificaciones

Esta es una página estática que genera una tabla de calificaciones basada en el número total de aciertos en un examen o evaluación.

## Uso

1. Introduce el número total de aciertos en el campo de entrada.
2. Haz clic en el botón "Generar Tabla".
3. La tabla de calificaciones se mostrará a continuación.

## Despliegue

La página está desplegada en GitHub Pages en la siguiente URL: [https://<tu-usuario>.github.io/calificaciones/](https://<tu-usuario>.github.io/calificaciones/)
